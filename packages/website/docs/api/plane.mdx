# Plane

The plane module designates particular axes to their respective visulizers and navigators. Use the [Axis](./axis.mdx) context to declares all axes used in your application. See the [Basic Spectrogram](../examples/basic-spectrogram.mdx) demo for guidance.

### Components

```ts
function Provider(props: ProviderProps): JSX.Element
```

### Functions

```ts
function useContext(): Context
```

### Types

```ts
type Context = {
  xaxis: Axis.Axis
  yaxis: Axis.Axis
}

type ProviderProps = {
  children: React.ReactNode
  xaxis: string
  yaxis: string
}
```
