Runtime and Dependencies​

After installing the bun JavaScript runtime and package manager, install the development dependencies.

$ bun install
bun install v1.1.29 (6d43b366)

1116 packages installed [3.77s]

Blocked 3 postinstalls. Run `bun pm untrusted` for details.

Examples​

To run a starter project, change to the example's directory and run the dev script.

$ cd examples/advanced-annotation
$ bun run dev

The example webserver will display the connection URL. In the default case, http://localhost:5173.

Release​

The Specviz package is released using the following process.

StepDescriptionCommand
1.Run the linter and formatterbun run check
2.Run the Specviz build scriptbun run --cwd packages/specviz build
3.Run the package testsbun run test
4.Run the TypeScript compilerbun run tsc
5.Update the version fieldvim packages/specviz/package.json
6.Commit the changesgit commit -m "version X.Y.Z"
7.Tag the commit with versiongit tag X.Y.Z
8.Push the commit and taggit push && git push --tags
9.Run the bun pm pack commandbun pm --cwd packages/specviz pack
10.Run the bun publish commandbun publish packages/specviz/meridian-analytics-specviz-X.Y.Z.tgz

To publish, you will need to be logged into npm.

$ bunx npm login

To see the current logged in user:

$ bun pm whoami
